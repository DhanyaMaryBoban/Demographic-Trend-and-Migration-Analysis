# Demographic-Trend-and-Migration-Analysis

## 👩‍💻 Project Overview
This project analyzes population trends and net migration patterns across five South Asian countries: India, Pakistan, Bangladesh, Afghanistan, Sri Lanka. This include exploratory data analysis and Power BI visualizations to uncover demographic insights, migration flows, and correlations between population size and migration.

## 📂 Dataset
- Records: 320
- Attributes: 4 features

  ## 🛠️ Tools Used
- Language: Python
- Libraries: Pandas, NumPy, Matplotlib, Seaborn
- Visuals: Power BI

  ## 🧹 Data Cleaning and Preprocessing
- Steps performed:
    - Make sure there is no missing values
    - Checking whether there is no duplicates
    - Correcting datatypes for consistency
 
  ## 📊 Exploratory Data Analysis (EDA)
- Computed year over year growth rates to track changes over time.
- Identified outliers in net migration and removed it by using IQR method.
- Used scatter plots to identify patterns and correlations of population and migration.

  ## 🔍 Key Insights
- India is the most populous country, showing steady growth from 2019–2023.
- All countries exhibit negative net migration, meaning more people are leaving than entering.
- Top emigrant sources:  Bangladesh, Pakistan and  India.
- Afghanistan showed a brief rise in immigration during the 1980s.
- Weak negative correlation (-0.123) between population size and net migration, suggesting external factors (economic, political, social) drive migration more than population size.

## 📊 Power BI Insights
- Net Migration (2019–2023): Regions experienced a net loss of 2 million people.
- Population Decline: South Asian countries saw an average 2.49% annual decrease, likely due to outward migration, lower birth rates, or data inconsistencies.
- Anomalies: A spike in 2021 population data requires validation.
- Population Distribution: Map visualization confirms India as the largest population, followed by Bangladesh and Pakistan.

## 📌 Conclusion
South Asia is a major source of emigrants globally, with population growth driven mainly by natural increase rather than immigration; migration flows are shaped by external pressures rather than population size, and this project demonstrates analytical strength, clear storytelling, and effective data visualization to uncover meaningful insights into population and migration trends.
